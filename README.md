## Анализатор страниц

Cайт, выполняющий проверку указанной веб-страницы на SEO пригодность. 

#### Требования:

- Composer
- PHP (^8.0)
- Node.js & npm

#### Установка:

$ git clone https://github.com/hallernsk/php-project-lvl3.git

$ cd php-project-lvl3

$ make setup

#### Запуск:

$ make start
